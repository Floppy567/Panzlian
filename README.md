# Panzlian Language Specifications:
Panzlian is a constructed language alien to most human scripts, non linear (2D structure).
This repository contains a formal guide on how to understand this script, both the formal and colloquial Panzlian.
Status: early specification, unstable.

## Design Principles
- Non-linear redability
- Strong and weak internal links
- Spatial grammar
- Machine friendly structure
- position based grouping semantics
- top right to bottom left reading
